Aulas De Banco De Dados

Neste Período conhecemos sobre o BDO ou Banco de Dados, onde criamos tabelas, e dentro dessas tabelas, seus atributos.

Daí então, utilizamos um aplicativo chamado MAMP(no Windows usamos o XAMP), e começamos a criar nossos bancos.

Animação

Depois, começamos a fazer consltas no SQL, via JOIN, onde juntamos as tabelas para consultas maiores.
