BootStrap

O que é o Bootstrap ?

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

Formas de Instalar

Compilado : Para configurar o Bootstrap compilado, deve-se fazer o download do arquivo compactado, e então extraí-lo na pasta onde está localizada a sua aplicação web.

Via CDN :Para utilizá-la, basta o código necessário para importar os arquivos necessários.

Medias

As media queries são uma parte fundamental do design responsivo na web. Elas são usadas em CSS (Cascading Style Sheets) para aplicar estilos diferentes a um documento com base em características específicas do dispositivo ou da tela. Essas características podem incluir largura e altura da tela, resolução da tela e muito mais.
